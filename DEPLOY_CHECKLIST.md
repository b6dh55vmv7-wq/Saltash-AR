## 📋 Deployment Steps

### Step 1: Create GitHub Account
- [ ] Go to [github.com](https://github.com)
- [ ] Click "Sign up"
- [ ] Enter email address
- [ ] Create password
- [ ] Choose username (e.g., `your-username`)
- [ ] Verify email
- **Time: 2 minutes**

### Step 2: Create New Repository
- [ ] Click "+" icon (top right)
- [ ] Select "New repository"
- [ ] **Repository name:** `saltash-webar-tour`
- [ ] **Description:** "Saltash WebAR Historical Tour"
- [ ] **Visibility:** Public
- [ ] Leave "Initialize with README" unchecked
- [ ] Click "Create repository"
- **Time: 1 minute**

### Step 3: Upload Files
- [ ] Click "Add file" → "Upload files"
- [ ] Drag and drop all files from the project folder:
  - [ ] `index.html` (home page)
  - [ ] `tour.html` (AR experience)
  - [ ] `markers.html` (markers guide)
  - [ ] `README.md` (documentation)
  - [ ] `DEPLOYMENT.md` (deployment guide)
  - [ ] `QUICK_START.md` (quick reference)
  - [ ] `GITHUB_DEPLOYMENT.md` (this guide)
  - [ ] `netlify.toml` (configuration)
  - [ ] `vercel.json` (configuration)
  - [ ] `package.json` (metadata)
  - [ ] `.gitignore` (git configuration)
- [ ] Add commit message: "Initial commit: Saltash WebAR Historical Tour"
- [ ] Click "Commit changes"
- **Time: 1 minute**

### Step 4: Enable GitHub Pages
- [ ] Go to your repository
- [ ] Click "Settings" (top right)
- [ ] In left sidebar, click "Pages"
- [ ] Under "Build and deployment":
  - [ ] Source: "Deploy from a branch"
  - [ ] Branch: "main" (or "master")
  - [ ] Folder: "/ (root)"
- [ ] Click "Save"
- **Time: 1 minute**

### Step 5: Wait for Deployment
- [ ] Go back to main repository page
- [ ] Look for yellow/orange dot next to "Deployments"
- [ ] Wait 1-2 minutes for it to turn green ✅
- [ ] Your site is now live!
- **Time: 2 minutes**

### Step 6: Access Your Website
- [ ] Your URL: `https://YOUR-USERNAME.github.io/saltash-webar-tour`
- [ ] Replace `YOUR-USERNAME` with your actual GitHub username
- [ ] Test on mobile device
- [ ] Verify all pages load correctly
- **Time: 1 minute**

## 🆘 Troubleshooting

### Site not showing
- Refresh page (Ctrl+F5)
- Check Pages settings (should be "Deploy from a branch")
- Verify branch name (main or master)
- Wait up to 5 minutes

### Camera not working
- Ensure HTTPS (GitHub provides this)
- Grant camera permissions
- Try different browser
- Use smartphone instead of desktop

### AR markers not detecting
- Ensure good lighting
- Print markers at larger size (15cm+)
- Hold device 20-50cm from marker
- Try different device/browser
