# Quick Start - Deploy in 2 Minutes ⚡

## Choose Your Platform

### 🥇 Netlify (Recommended - Easiest)

**Time: 2 minutes | Cost: Free**

1. Go to **[netlify.com](https://netlify.com)**
2. Click **"Add new site"** → **"Deploy manually"**
3. **Drag and drop** this entire folder
4. **Done!** Your site is live with a URL like `https://your-site.netlify.app`

---

### 🥈 Vercel (Fast & Reliable)

**Time: 2 minutes | Cost: Free**

1. Go to **[vercel.com](https://vercel.com)**
2. Click **"New Project"** → **"Continue with File Upload"**
3. **Upload** this folder
4. **Done!** Your site is live with a URL like `https://your-project.vercel.app`

---

### 🥉 GitHub Pages (Free & Open)

**Time: 5 minutes | Cost: Free**

1. Create a **[GitHub](https://github.com)** account
2. Create a **new repository** named `saltash-webar-tour`
3. **Upload all files** from this folder
4. Go to **Settings** → **Pages**
5. Select **"Deploy from a branch"** → **"main"**
6. **Done!** Your site is live at `https://yourusername.github.io/saltash-webar-tour`

## 🔗 Next Steps

### Add a Custom Domain (Optional)

Want `saltash-webar-tour.com` instead of the default URL?

1. **Buy a domain** at [Namecheap](https://namecheap.com) or [GoDaddy](https://godaddy.com)
2. **Connect to your hosting:**
   - **Netlify:** Site Settings → Domain Management → Add Custom Domain
   - **Vercel:** Project Settings → Domains → Add
   - **GitHub Pages:** Settings → Pages → Custom Domain

3. **Update DNS** at your registrar (instructions provided by platform)
4. **Wait 24-48 hours** for DNS to propagate
